# DBMS Project

Development Status: Under development

## Tech Stack

### Frontend

- React Native
- Typescript
- URQL
- Chakra UI
- Framer Motion

### Backend

- NodeJS
- Typescript
- Apollo and Express Apollo
- GraphQL
- Prisma
- PostgreSQL
- Redis

## How to setup the project

### Frontend Setup

Run

```bash
npm i # to install dependencies
npm start # to run frontend server
```

### Backend Setup

Run
```bash
npm i
npm run dev # to start development server
```
